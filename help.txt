######  Подсказки по гиту:

Алгоритм:
1) Создаем публичный репозиторий на Github
2) Клонируем репозиторий на комп:
git clone "https://github.com/..."
3) Изменяем файлы и коммитим:
git commit -m "..."
4) Заливаем изменения на сервер:
git push

Если нужно подтянуть изменения из интернета на комп:
git pull


######  Этапы создания бота:

1) Установка Виртуального окружения
python3 -m venv venv

2) Активируем окружение
source venv/bin/activate

3) Смотрим установленные библиотеки:
pip list
